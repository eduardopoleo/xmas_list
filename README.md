# Eduardo's Christmas List

A responsive static website for displaying Christmas wishlist items.

## Features

- Responsive design that works on desktop, tablet, and mobile
- Stylized Christmas-themed design
- Each gift item includes:
  - Name
  - Image
  - Link to product
  - Description

## Customization

Edit `index.html` to add your own gift items. For each gift, update:
- Image URL in the `<img>` tag
- Gift name in the `<h2>` tag
- Description in the `<p class="gift-description">` tag
- Product link in the `<a href="">` tag

## Deploying to Vercel

1. Install Vercel CLI (if not already installed):
   ```bash
   npm install -g vercel
   ```

2. Deploy to Vercel:
   ```bash
   vercel
   ```

Or simply push to a GitHub repository and connect it to Vercel through their web interface.

## Local Development

Simply open `index.html` in your browser to view the site locally.
